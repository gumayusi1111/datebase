# 个人数据库项目

这是一个简单的个人数据库应用，允许用户存储和检索各种类型的数据（文字、图片、语音），并通过AI接口进行智能查询。

## 项目流程

### 已完成
- [x] 创建React项目基础结构
- [x] 实现本地存储服务
- [x] 创建数据输入组件
- [x] 创建数据列表组件
- [x] 实现搜索功能
- [x] 创建AI查询接口（使用DeepSeek-R1 API）
- [x] 实现基本样式

### 当前进度
