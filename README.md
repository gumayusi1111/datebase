# 个人数据库项目

这是一个简单的个人数据库应用，允许用户存储和检索各种类型的数据（文字、图片、语音），并通过智能查询功能访问数据。

## 功能特点

- ✅ 用户注册和登录系统
- ✅ 添加带标签的数据记录
- ✅ 支持文本、图片和音频数据
- ✅ 地理位置标记
- ✅ 智能搜索和查询
- ✅ 标签管理和统计
- ✅ 响应式设计，适配移动设备

## 技术栈

- **前端**: React, Axios, CSS3
- **后端**: Node.js, Express
- **数据存储**: 文件系统 (JSON)
- **认证**: JWT (JSON Web Tokens)
- **智能查询**: 本地模拟 AI 响应
